# 🌍 Environmental Impact Dashboard

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://seu-usuario-environmental-dashboard.streamlit.app)

Dashboard interativo que unifica **10 projetos de pesquisa ambiental** (2016-2026) utilizando técnicas de **Big Data Analytics**, **Ciência de Dados** e **Machine Learning** para quantificar o impacto ecológico.

## 📊 Funcionalidades

- **Métricas Globais:** Carbono poupado (tCO₂e) e energia produzida (MWh) desde Agosto/2016.
- **Visualizações Interativas:** Gráficos Plotly (linha, área, barra, mapa, pizza, 3D).
- **Processamento Big Data:** Simulação com PySpark (agregações, top N, escalável).
- **Modelos de Data Science:**
  - Previsão de séries temporais (Holt-Winters)
  - Clustering de projetos (K-Means)
  - Feature Importance (Random Forest)
- **Layout Responsivo:** Abas organizadas, cards de métricas, design limpo.
